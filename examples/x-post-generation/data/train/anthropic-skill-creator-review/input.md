# テーマ
Anthropic公式の「skill-creator」のフィードバック機能のレビュー

# やったこと / 素材
- Anthropic公式の「skill-creator」を試用した
- この公式Skillsには「作る」だけでなく「評価・改善する」ステップがある
- 評価対象：成果物の質、使用トークン数、タスクにかかった時間
- 評価アプローチ：「AIと人間のハイブリッド評価」
  - AIが事前に自動で設定した基準を満たしているか評価
  - 人間がAIの評価後に「本当に良いか」を判断し自然言語でフィードバック
  - AIがある程度スコアをつけ、人間が最終的な質を見る分担
- 改善は一回では終わらず、何度でも人間がフィードバックを与え続け改善し続ける設計
- 以前に自分が「人間が過去に作成した正解データを用いてSkillsを自動で最適化する」という記事を書いた
- 対比：Anthropic公式は手動フィードバックによる評価、自分の記事は自動評価を目指す
- 自分が実験しているのは「人間がフィードバックを行う箇所のAIによる自動化」
- 事前に詳細な評価指標を用意し、実際に改善中のSkillsを走らせることで人間が評価する必要性をなくす狙い
- Xの記事作成のSkillsを作ってみた様子の画像あり
